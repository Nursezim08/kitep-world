# Руководство по использованию функции "Забыли пароль"

## Для пользователей

### Как восстановить пароль:

1. **Перейдите на страницу входа** (`/login`)
   - Нажмите на ссылку "Забыли пароль?" под полем ввода пароля

2. **Введите ваш email** (`/forgot-password`)
   - Введите email, который вы использовали при регистрации
   - Нажмите "Отправить код"
   - Если email не зарегистрирован, появится сообщение: "Эта почта не зарегистрирована на нашем сайте"
   - Если это Google аккаунт, появится сообщение о необходимости использовать вход через Google

3. **Введите код из email** (`/reset-password/verify`)
   - Проверьте вашу почту
   - Введите 6-значный код из письма
   - Код действителен 10 минут
   - Можно вставить код из буфера обмена
   - Если код неверный или истек, запросите новый

4. **Установите новый пароль** (`/reset-password/new`)
   - Введите новый пароль
   - Повторите новый пароль
   - Пароль должен соответствовать требованиям:
     - Минимум 8 символов
     - Хотя бы одна заглавная буква
     - Хотя бы одна строчная буква
     - Хотя бы одна цифра
   - Нажмите "Сохранить пароль"

5. **Готово!**
   - Вы будете автоматически перенаправлены в профиль
   - Теперь можете войти с новым паролем

### API Endpoints:

#### POST `/api/auth/forgot-password`
**Request:**
```json
{
  "email": "user@example.com"
}
```

**Response (Success):**
```json
{
  "success": true,
  "message": "Код для сброса пароля отправлен на вашу почту"
}
```

**Response (Error):**
```json
{
  "error": "Эта почта не зарегистрирована на нашем сайте"
}
```

#### POST `/api/auth/verify-reset-code`
**Request:**
```json
{
  "email": "user@example.com",
  "code": "123456"
}
```

**Response (Success):**
```json
{
  "success": true,
  "message": "Код подтвержден"
}
```

**Response (Error):**
```json
{
  "error": "Неверный код"
}
```

#### POST `/api/auth/reset-password`
**Request:**
```json
{
  "email": "user@example.com",
  "code": "123456",
  "newPassword": "NewPassword123"
}
```

**Response (Success):**
```json
{
  "success": true,
  "message": "Пароль успешно изменен"
}
```

**Response (Error):**
```json
{
  "error": "Пароль должен содержать минимум 8 символов"
}
```

### База данных:

**Таблица `password_resets`:**
```sql
CREATE TABLE "password_resets" (
    "id" UUID PRIMARY KEY,
    "email" VARCHAR(255) NOT NULL,
    "code" VARCHAR(6) NOT NULL,
    "expires_at" TIMESTAMP(6) NOT NULL,
    "used" BOOLEAN NOT NULL DEFAULT false,
    "created_at" TIMESTAMP(6) NOT NULL DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX "password_resets_email_code_idx" 
ON "password_resets"("email", "code");
```

### Безопасность:

- ✅ Коды действительны только 10 минут
- ✅ Коды одноразовые (флаг `used`)
- ✅ Старые неиспользованные коды удаляются
- ✅ Валидация пароля на сервере
- ✅ Защита от сброса пароля Google аккаунтов
- ✅ Хеширование паролей с bcrypt
- ✅ Проверка существования пользователя

### Тестирование:

1. **Без SMTP настроек:**
   - Коды выводятся в консоль сервера
   - Удобно для разработки

2. **С SMTP настройками:**
   - Коды отправляются на реальную почту
   - Красивый HTML шаблон письма

### Настройка SMTP (опционально):

В файле `.env`:
```env
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
```

Для Gmail используйте App Password:
1. Включите 2FA в Google аккаунте
2. Создайте App Password в настройках безопасности
3. Используйте этот пароль в `SMTP_PASS`

## Возможные ошибки и решения

### "Эта почта не зарегистрирована на нашем сайте"
- Проверьте правильность email
- Зарегистрируйтесь, если еще не создали аккаунт

### "Этот аккаунт зарегистрирован через Google"
- Используйте кнопку "Войти через Google" на странице логина
- Восстановление пароля недоступно для Google аккаунтов

### "Неверный код"
- Проверьте правильность введенного кода
- Код чувствителен к регистру (только цифры)

### "Код истек"
- Запросите новый код на странице `/forgot-password`
- Коды действительны только 10 минут

### "Пароли не совпадают"
- Убедитесь, что оба поля содержат одинаковый пароль

### Ошибки валидации пароля:
- "Пароль должен содержать минимум 8 символов"
- "Пароль должен содержать хотя бы одну заглавную букву"
- "Пароль должен содержать хотя бы одну строчную букву"
- "Пароль должен содержать хотя бы одну цифру"
