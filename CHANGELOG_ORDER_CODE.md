### Технические детали

#### Алгоритм извлечения кода

```typescript
const getOrderCode = (orderNumber: string) => {
  // 1. Убираем дефисы
  // 2. Убираем все нецифровые символы
  // 3. Берем последние 5 цифр
  const digits = orderNumber.replace(/-/g, '').replace(/\D/g, '');
  return digits.slice(-5);
};
```

**Примеры:**
- `ORD-1779710728185-808` → `1779710728185808` → `85808`
- `ORD-1234567890123-456` → `1234567890123456` → `23456`
