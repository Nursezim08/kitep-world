# Инструкция по заполнению базы данных тестовыми данными

## Описание

Seed скрипт создаёт:
- **10 категорий** товаров (Канцелярия, Книги, Тетради, Ручки, Карандаши, Папки и файлы, Краски и кисти, Рюкзаки, Пеналы, Офисная техника)
- **По 10 товаров** в каждой категории (всего 100 товаров)
- **Переводы** на русском и кыргызском языках для всех категорий и товаров
- **4 изображения** для каждого товара (placeholder изображения)

## Установка зависимостей

Сначала установите необходимые зависимости:

```bash
npm install
```

## Запуск seed скрипта

### Вариант 1: Через npm скрипт (рекомендуется)

```bash
npm run db:seed
```

### Вариант 2: Через Prisma CLI

```bash
npx prisma db seed
```

### Вариант 3: Напрямую через tsx

```bash
npx tsx prisma/seed.ts
```

## Что будет создано

### Категории (10 шт):

1. **Канцелярия** - степлеры, дыроколы, ножницы, клей, скотч и т.д.
2. **Книги** - классическая литература, учебники, энциклопедии
3. **Тетради** - школьные тетради, блокноты, дневники
4. **Ручки** - шариковые, гелевые, автоматические, наборы
5. **Карандаши** - простые, цветные, механические
6. **Папки и файлы** - папки-скоросшиватели, файлы, обложки
7. **Краски и кисти** - акварель, гуашь, кисти, палитры
8. **Рюкзаки** - школьные, спортивные, туристические
9. **Пеналы** - школьные, силиконовые, с наполнением
10. **Офисная техника** - калькуляторы, ламинаторы, печати

### Товары (100 шт):

Каждый товар содержит:
- ✅ Уникальный SKU (например: КАН-0001, КНИ-0002)
- ✅ Название на русском и кыргызском
- ✅ Описание на русском и кыргызском
- ✅ Бренд
- ✅ Цена (от 20 до 12000 сом)
- ✅ Статус: active
- ✅ 4 placeholder изображения

## Очистка данных

⚠️ **ВНИМАНИЕ:** Скрипт автоматически удаляет все существующие категории и товары перед созданием новых!

Если вы хотите сохранить существующие данные, закомментируйте следующие строки в `prisma/seed.ts`:

```typescript
// await prisma.productImage.deleteMany();
// await prisma.productTranslation.deleteMany();
// await prisma.productAttribute.deleteMany();
// await prisma.product.deleteMany();
// await prisma.categoryTranslation.deleteMany();
// await prisma.category.deleteMany();
```

## Просмотр данных

### В админке:

1. Откройте админку: `http://localhost:3000/admin/login`
2. Войдите как администратор
3. Перейдите в раздел "Категории" или "Товары"

### В Prisma Studio:

```bash
npx prisma studio
```

## Изменение данных

Если вы хотите изменить данные seed скрипта:

1. Откройте файл `prisma/seed.ts`
2. Измените массивы `categories` или `productTemplates`
3. Запустите seed скрипт заново

## Troubleshooting

### Ошибка: "Cannot find module 'tsx'"

Установите зависимости:
```bash
npm install
```

### Ошибка: "Database connection failed"

Проверьте переменную `DATABASE_URL` в файле `.env`

### Ошибка: "Foreign key constraint failed"

Убедитесь, что база данных пустая или закомментируйте строки очистки данных

## Дополнительно

### Изменение изображений

По умолчанию используются placeholder изображения. Чтобы использовать реальные изображения:

1. Загрузите изображения в S3
2. Замените URL в массиве `productImages` в `prisma/seed.ts`

### Добавление новых категорий

Добавьте новую категорию в массив `categories`:

```typescript
{
  nameRu: 'Новая категория',
  nameKg: 'Жаңы категория',
  image: 'https://example.com/image.png',
}
```

И создайте шаблоны товаров в `productTemplates`:

```typescript
'Новая категория': [
  { nameRu: 'Товар 1', nameKg: 'Товар 1', brand: 'Brand', price: 100 },
  // ... ещё 9 товаров
]
```
