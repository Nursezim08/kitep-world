# 📋 Юридические страницы - Краткое резюме

## ✨ Что реализовано

Созданы полноценные юридические страницы в соответствии с законодательством Кыргызской Республики:

### 🛡️ Политика конфиденциальности (`/privacy`)
- **12 разделов** с полной информацией о обработке персональных данных
- **Соответствие** Закону КР "О персональных данных" (14.04.2008 № 58)
- **Права пользователя:** доступ, исправление, удаление, ограничение, возражение, портируемость
- **Защита данных:** шифрование SSL/TLS, ограничение доступа, мониторинг безопасности
- **Размер:** 18.5 KB, ~400 строк кода

### 📄 Условия использования (`/terms`)
- **13 разделов** с полными условиями использования сервиса
- **Соответствие** Закону КР "О защите прав потребителей"
- **Ключевые пункты:** регистрация, заказы, оплата, возврат (14 дней), гарантии, споры
- **Претензионный порядок:** 30 дней, подсудность КР
- **Размер:** 25.6 KB, ~570 строк кода

## 🎯 Быстрый старт

### Команды для замены данных

**macOS/Linux:**
```bash
# Заменить название компании
find app/\(legal\) -type f -exec sed -i '' 's/\[Название юридического лица\]/ОсОО "Нур-Китеп"/g' {} +

# Заменить ИНН
find app/\(legal\) -type f -exec sed -i '' 's/\[ИНН\]/12345678901234/g' {} +

# Заменить адрес
find app/\(legal\) -type f -exec sed -i '' 's/\[Юридический адрес\]/г. Бишкек, ул. Киевская 147/g' {} +
```

**Windows (PowerShell):**
```powershell
# Заменить название компании
Get-ChildItem -Path "app\(legal)" -Recurse -File | ForEach-Object {
    (Get-Content $_.FullName) -replace '\[Название юридического лица\]', 'ОсОО "Нур-Китеп"' | 
    Set-Content $_.FullName
}

# Заменить ИНН
Get-ChildItem -Path "app\(legal)" -Recurse -File | ForEach-Object {
    (Get-Content $_.FullName) -replace '\[ИНН\]', '12345678901234' | 
    Set-Content $_.FullName
}
```

### ✅ Что делать
- Регулярно обновляйте документы (минимум раз в год)

### ❌ Чего избегать
- Не публикуйте с плейсхолдерами `[...]`
- Не копируйте документы из других стран без адаптации
- Не игнорируйте запросы пользователей об удалении данных
- Не забывайте обновлять дату последнего изменения
