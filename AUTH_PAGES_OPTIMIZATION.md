# Оптимизация страниц входа и регистрации

## Проблема

Страницы входа и регистрации были слишком большими, требовали скролла для просмотра всех элементов (особенно кнопки "Войти через Google").

## Решение

Уменьшены размеры и отступы всех элементов для размещения контента на одном экране без скролла.

## Изменения

### Общие изменения

| Элемент | Было | Стало | Изменение |
|---------|------|-------|-----------|
| Padding контейнера | `p-8` (32px) | `p-6` (24px) | -25% |
| Padding страницы | `px-4` | `px-4 py-6` | +вертикальные отступы |
| Размер логотипа | `w-16 h-16` | `w-12 h-12` | -25% |
| Размер текста логотипа | `text-xl` | `text-lg` | -1 размер |
| Gap логотипа | `gap-2` | `gap-1` | -50% |
| Margin логотипа | `mb-6` | `mb-4` | -33% |

### Заголовки

| Элемент | Было | Стало | Изменение |
|---------|------|-------|-----------|
| H1 размер | `text-3xl` | `text-2xl` | -1 размер |
| H1 margin | `mb-2` | `mb-1` | -50% |
| Описание размер | `text-base` | `text-sm` | -1 размер |
| Секция margin | `mb-8` | `mb-5` | -37.5% |

### Форма

| Элемент | Было | Стало | Изменение |
|---------|------|-------|-----------|
| Space между полями | `space-y-6` | `space-y-4` (login)<br>`space-y-3.5` (register) | -33% / -42% |
| Label margin | `mb-2` | `mb-1.5` | -25% |
| Input padding | `py-3` | `py-2.5` | -17% |
| Input размер текста | `text-base` | `text-sm` | -1 размер |
| Button padding | `py-3` | `py-2.5` | -17% |
| Button размер текста | `text-base` | `text-sm` | -1 размер |

### Ошибки

| Элемент | Было | Стало | Изменение |
|---------|------|-------|-----------|
| Error padding | `p-4` | `p-3` | -25% |
| Error margin | `mb-6` | `mb-4` | -33% |

### Разделитель и Google кнопка

| Элемент | Было | Стало | Изменение |
|---------|------|-------|-----------|
| Секция margin | `mt-6` | `mt-4` | -33% |
| Google button margin | `mt-6` | `mt-4` | -33% |
| Google button padding | `py-3` | `py-2.5` | -17% |
| Google button размер текста | `text-base` | `text-sm` | -1 размер |
| Google icon размер | `size={24}` | `size={22}` | -8% |

### Нижний текст

| Элемент | Было | Стало | Изменение |
|---------|------|-------|-----------|
| Margin | `mt-8` | `mt-5` | -37.5% |

## Рекомендации

### Для дальнейшей оптимизации

1. **Убрать описание под заголовком** (если нужно еще больше места)
2. **Объединить поля** (например, email и пароль в одну секцию)
3. **Использовать tabs** для переключения вход/регистрация

### Для сохранения читаемости

1. ✅ Не уменьшать размер текста меньше `text-sm`
2. ✅ Не уменьшать padding полей меньше `py-2`
3. ✅ Сохранять достаточные отступы между элементами
