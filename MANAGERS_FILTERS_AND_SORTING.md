# Система фильтрации и сортировки менеджеров

## Реализованные возможности

### 🔍 Поиск
- **Текстовый поиск** по:
  - Полному имени менеджера
  - Email адресу
  - Номеру телефона
- Поиск работает в реальном времени
- Регистронезависимый поиск

### 🎯 Фильтры

#### 1. Фильтр по статусу
- **Все статусы** (по умолчанию)
- **Активен** - только активные менеджеры
- **Неактивен** - неактивные менеджеры
- **Заблокирован** - заблокированные менеджеры

#### 2. Фильтр по филиалу
- **Все филиалы** (по умолчанию)
- **Не назначен** - менеджеры без филиала
- Список всех доступных филиалов с городами

### 📊 Сортировка

#### Поля для сортировки:
1. **По дате создания** (по умолчанию)
2. **По имени** - алфавитный порядок
3. **По Email** - алфавитный порядок
4. **По филиалу** - алфавитный порядок названий филиалов

#### Порядок сортировки:
- **По возрастанию** (A-Z, 0-9, старые → новые)
- **По убыванию** (Z-A, 9-0, новые → старые)
- Переключение одной кнопкой с иконками стрелок

### 🎨 UI/UX Особенности

#### Кнопка фильтров
- Иконка фильтра
- Подсветка фиолетовым цветом при активных фильтрах
- Счетчик активных фильтров (бейдж с числом)
- Показывает/скрывает панель фильтров

#### Панель фильтров
- Раскрывающаяся панель под поиском
- Адаптивная сетка (1-4 колонки в зависимости от экрана)
- Кнопка "Сбросить" для очистки всех фильтров
- Визуальное отображение активных фильтров с возможностью удаления

#### Сортировка в таблице
- Кликабельные заголовки колонок
- Иконки стрелок показывают направление сортировки
- Только для колонок: Менеджер, Email, Филиал
- Плавная анимация при наведении

#### Счетчик результатов
- Показывает количество отфильтрованных менеджеров
- Показывает общее количество менеджеров
- Формат: "Показано X из Y менеджеров"

#### Активные фильтры
- Отображаются как теги под панелью фильтров
- Каждый тег можно удалить кнопкой X
- Показывают текущие значения фильтров
- Фиолетовый цвет для визуального выделения

### 💡 Логика работы

#### Приоритет фильтрации:
1. Текстовый поиск
2. Фильтр по статусу
3. Фильтр по филиалу
4. Сортировка результатов

#### Комбинирование фильтров:
- Все фильтры работают одновременно (AND логика)
- Поиск + статус + филиал = пересечение всех условий
- Результаты обновляются мгновенно

#### Сброс фильтров:
- Кнопка "Сбросить" возвращает все к значениям по умолчанию:
  - Очищает поиск
  - Статус: "Все статусы"
  - Филиал: "Все филиалы"
  - Сортировка: "По дате создания"
  - Порядок: "По убыванию"

### 🎯 Состояния интерфейса

#### Пустой список
- Показывает "Нет менеджеров" если база пуста
- Показывает "Менеджеры не найдены" если есть активные фильтры/поиск

#### Загрузка
- Показывает "Загрузка..." пока данные загружаются

#### С результатами
- Отображает таблицу с менеджерами
- Показывает счетчик результатов

### 📱 Адаптивность

#### Мобильные устройства (< 640px):
- Поиск и кнопка фильтров в колонку
- Панель фильтров: 1 колонка
- Таблица с горизонтальным скроллом

#### Планшеты (640px - 1024px):
- Поиск и кнопка фильтров в ряд
- Панель фильтров: 2 колонки

#### Десктоп (> 1024px):
- Полная ширина
- Панель фильтров: 4 колонки
- Оптимальное использование пространства

### 🔧 Технические детали

#### Состояния компонента:
```typescript
const [searchQuery, setSearchQuery] = useState('');
const [statusFilter, setStatusFilter] = useState<string>('all');
const [branchFilter, setBranchFilter] = useState<string>('all');
const [sortBy, setSortBy] = useState<'name' | 'email' | 'date' | 'branch'>('date');

```

#### Функции:
- `filteredManagers` - применяет поиск и фильтры
- `sortedManagers` - применяет сортировку к отфильтрованным данным
- `handleSort()` - обрабатывает клики по заголовкам таблицы
- `resetFilters()` - сбрасывает все фильтры
- `activeFiltersCount` - подсчитывает активные фильтры

#### Производительность:
- Фильтрация и сортировка на клиенте (быстро для небольших списков)
- Мемоизация не требуется для текущего объема данных
- При необходимости можно перенести на сервер для больших объемов

## Использование

1. **Быстрый поиск**: Введите текст в поле поиска
2. **Открыть фильтры**: Нажмите кнопку "Фильтры"
3. **Выбрать фильтры**: Выберите статус и/или филиал
4. **Настроить сортировку**: Выберите поле и порядок
5. **Сортировка в таблице**: Кликните на заголовок колонки
6. **Сбросить**: Нажмите "Сбросить" для очистки всех фильтров
7. **Удалить фильтр**: Кликните X на теге активного фильтра
