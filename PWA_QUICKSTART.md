# PWA Quick Start - Nur-Kitep

## 🚀 Быстрый старт

### 1. Установите зависимости:
```bash
npm install
```

### 2. Сгенерируйте иконки (уже сделано):
```bash
npm run generate:icons
```

### 3. Соберите production версию:
```bash
npm run build
```

### 4. Запустите production сервер:
```bash
npm run start
```

### 5. Откройте приложение:
```
http://localhost:3000
```

## ✅ Проверка PWA

### В браузере:
1. Откройте **Chrome DevTools** (F12)
2. Перейдите во вкладку **Application**
3. Проверьте:
   - ✅ **Manifest** загружен
   - ✅ **Service Worker** активирован
   - ✅ **Cache Storage** содержит файлы

### Lighthouse:
1. DevTools → **Lighthouse**
2. Выберите **Progressive Web App**
3. Нажмите **Analyze page load**
4. Ожидаемая оценка: **100/100**

### Оффлайн тест:
1. DevTools → **Network** → ✅ **Offline**
2. Обновите страницу (F5)
3. Должна открыться страница `/offline`

## 📱 Установка PWA

### Android (Chrome):
1. Откройте сайт
2. Нажмите **⋮** → **Установить приложение**
3. Или используйте баннер внизу

### iOS (Safari):
1. Откройте сайт
2. Нажмите **Поделиться** 📤
3. Выберите **На экран Домой**
4. Нажмите **Добавить**

### Desktop (Chrome):
1. Иконка **установки** в адресной строке
2. Или используйте баннер внизу
3. Приложение откроется в отдельном окне

## 🎯 Возможности

✅ **Работает оффлайн** - доступ к главным страницам без интернета
✅ **Устанавливается** - как нативное приложение на любое устройство
✅ **Обновляется** - автоматически проверяет и применяет обновления
✅ **Быстро** - кеширование ускоряет загрузку
✅ **Удобно** - иконка на главном экране, без адресной строки
