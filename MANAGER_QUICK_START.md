# Быстрый старт: Вход менеджера

## 🚀 За 5 минут

### Шаг 1: Создайте менеджера (Администратор)

```
1. Откройте /admin/login
2. Войдите как администратор
3. Перейдите в /admin/managers
4. Нажмите "Добавить менеджера"
5. Заполните:
   - ФИО: Иван Иванов
   - Email: manager@example.com
   - Телефон: +996 555 123 456
   - Пароль: password123
6. Сохраните
```

### Шаг 2: Назначьте на филиал (Администратор)

```
1. Перейдите в /admin/branches
2. Выберите филиал или создайте новый
3. В секции "Менеджеры" добавьте созданного менеджера
4. Сохраните
```

### Шаг 3: Войдите как менеджер

```
1. Откройте /manager/login
2. Введите:
   - Email: manager@example.com
   - Пароль: password123
3. Нажмите "Войти"
```

### Шаг 4: Проверьте email

```
В dev режиме (без SMTP):
- Код будет в консоли сервера
- Найдите строку: "Code: 123456"

В production (с SMTP):
- Проверьте почту manager@example.com
- Откройте письмо "Код входа для менеджера Nur-Kitep"
- Скопируйте 6-значный код
```

### Шаг 5: Введите код

```
1. На странице /manager/verify введите код
2. Нажмите "Подтвердить"
3. Вы будете перенаправлены на /manager/branch/[id]
```

## ✅ Готово!

Теперь менеджер может:
- Просматривать информацию о филиале
- Видеть список всех менеджеров
- Управлять операциями филиала

## 🔧 Настройка SMTP (для production)

Добавьте в `.env`:

```env
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
```

### Как получить App Password для Gmail:

1. Откройте https://myaccount.google.com/security
2. Включите двухфакторную аутентификацию
3. Перейдите в "App passwords"
4. Создайте новый пароль для приложения
5. Скопируйте пароль в `.env`

## 🐛 Решение проблем

### Код не приходит на email?
```
✓ Проверьте SMTP_USER и SMTP_PASS в .env
✓ В dev режиме код выводится в консоль
✓ Проверьте папку "Спам"
```

### "Вы не привязаны ни к одному филиалу"?
```
✓ Назначьте менеджера на филиал в админ-панели
✓ Проверьте таблицу branch_users в БД
```

### "Неверный email или пароль"?
```
✓ Проверьте правильность email
✓ Email должен быть в нижнем регистре
✓ Проверьте пароль (чувствителен к регистру)
```

### "Неверный код подтверждения"?
```
✓ Код действителен 10 минут
✓ Запросите новый код кнопкой "Отправить код повторно"
✓ Проверьте правильность ввода (только цифры)
```
