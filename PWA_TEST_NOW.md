## 📱 КАК ПРОТЕСТИРОВАТЬ

### 1️⃣ Production Build (ОБЯЗАТЕЛЬНО!)
```bash
npm run build
npm run start
```
**Важно**: Откройте `http://localhost:3000` в браузере

### 2️⃣ Проверка в Chrome (Desktop)
1. Откройте DevTools (F12)
2. Перейдите в **Application → Manifest**
3. Убедитесь:
   - ✅ Все 8 иконок видны
   - ✅ Background color: **#8b5cf6** (фиолетовый)
   - ✅ Name: "Nur-kitep"

**Должны увидеть иконки с фиолетовым фоном и белым логотипом!**

### 3️⃣ Тест на мобильном устройстве

#### 🤖 Android (Chrome)
1. **Разверните на домен** (Vercel/Netlify/ngrok)
   - localhost не работает полноценно на мобильных
2. Откройте сайт в Chrome
3. Появится баннер: **"Добавить Nur-kitep на главный экран"**
4. Установите
5. **Проверьте иконку** - должна быть с фиолетовым фоном!

#### 🍎 iOS (Safari)
1. Разверните на домен
2. Откройте в Safari
3. Нажмите **Поделиться** → **На экран «Домой»**
4. **Проверьте иконку** - фиолетовый фон!

**Примечание iOS**: Если старая иконка все еще показывается:
- Удалите PWA с главного экрана
- Очистите кеш Safari (Настройки → Safari → Очистить историю)
- Попробуйте снова

## ❓ Проблемы?

### Иконка не отображается в DevTools?
```bash
# Проверьте, что иконки созданы
ls public/icons/

# Должны быть 8 файлов:
# icon-72x72.png ... icon-512x512.png
```

### Иконка не появляется на телефоне?
1. Убедитесь, что сделали `npm run build` (не dev режим!)
2. Проверьте, что сайт открыт через **HTTPS** (не http://)
3. Разверните на реальный домен (не localhost)

### iOS показывает старую иконку?
- Удалите PWA
- Очистите кеш Safari
- Перезапустите Safari
- Попробуйте снова
