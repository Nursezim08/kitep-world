### 4. Через SQL запрос

Если используете PostgreSQL клиент:

```sql
-- Проверка категорий
SELECT COUNT(*) FROM categories WHERE status = 'active';
-- Должно быть: 10

-- Проверка товаров
SELECT COUNT(*) FROM products WHERE status = 'active';
-- Должно быть: 100

-- Проверка переводов категорий
SELECT COUNT(*) FROM category_translations;
-- Должно быть: 20

-- Проверка переводов товаров
SELECT COUNT(*) FROM product_translations;
-- Должно быть: 200

-- Проверка изображений
SELECT COUNT(*) FROM product_images WHERE status = 'active';
-- Должно быть: 400

-- Товары по категориям
SELECT 
  c.id,
  ct.name as category_name,
  COUNT(p.id) as products_count
FROM categories c
LEFT JOIN category_translations ct ON c.id = ct.category_id AND ct.locale = 'ru'
LEFT JOIN products p ON c.id = p.category_id
WHERE c.status = 'active'
GROUP BY c.id, ct.name
ORDER BY ct.name;
-- Каждая категория должна иметь 10 товаров
```

## Troubleshooting

### Не вижу категории в админке

1. Проверьте, что вы вошли как администратор (role = admin)
2. Проверьте в Prisma Studio, что данные есть
3. Проверьте консоль браузера на ошибки
4. Перезагрузите страницу (Ctrl+R)

### Не вижу товары

1. Проверьте фильтры (статус, категория)
2. Проверьте поиск (очистите поле поиска)
3. Проверьте в Prisma Studio
4. Проверьте консоль браузера

### Изображения не загружаются

Это нормально! Используются placeholder изображения с `via.placeholder.com`. Если сайт недоступен:
1. Замените URL в `prisma/seed.ts`
2. Запустите `npm run db:seed` заново

### Нет переводов на кыргызском

1. Проверьте таблицу `product_translations` в Prisma Studio
2. Должны быть записи с `locale = 'kg'`
3. Проверьте, что в коде используется правильный locale
