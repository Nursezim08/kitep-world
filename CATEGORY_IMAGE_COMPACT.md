# Компактная область загрузки изображения в модальных окнах категорий

## Обзор

Уменьшена область загрузки изображения в модальных окнах добавления и редактирования категорий для более компактного и аккуратного дизайна.

## Изменения

### Было:

```tsx
<div className="w-full aspect-square bg-gray-700 ...">
  {/* Квадратная область ~280px высота */}
</div>

<div className="md:col-span-2 flex flex-col justify-center space-y-3">
  {/* Поля названий центрированы вертикально */}
</div>
```

**Проблемы:**
- ❌ Область загрузки занимала слишком много места
- ❌ Высота ~280px была избыточной
- ❌ Много прокрутки в модальном окне
- ❌ Поля названий были центрированы, а не выровнены по верху

### Стало:

```tsx
<div className="w-full h-[120px] bg-gray-700 ...">
  {/* Фиксированная высота 120px */}
</div>

<div className="md:col-span-2 space-y-3">
  {/* Поля названий выровнены по верху */}
</div>
```

**Преимущества:**
- ✅ Компактная область загрузки (120px)
- ✅ Визуальное выравнивание с полями названий
- ✅ Экономия ~160px вертикального пространства (57%)
- ✅ Меньше прокрутки в модальном окне

## Визуальное выравнивание

### Структура:

```
┌─────────────────────────────────────────────────────┐
│ Изображение (PNG) *    Название (Русский) *         │
│ ┌─────────────────┐    ┌──────────────────────────┐ │
│ │                 │    │ Например: Канцелярия     │ │
│ │   [Upload Icon] │    └──────────────────────────┘ │
│ │   Перетащите    │                                 │
│ │   или кликните  │    Название (Кыргызча) *        │
│ │   PNG, макс 5MB │    ┌──────────────────────────┐ │
│ └─────────────────┘    │ Мисалы: Канцелярия       │ │
│                        └──────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

### Выравнивание:
- Верх области загрузки = верх поля "Название (Русский)"
- Низ области загрузки = низ поля "Название (Кыргызча)"
- Высота области загрузки = высота двух полей + отступ между ними

## Технические детали

### Размеры:

**Область загрузки:**
- Высота: `h-[120px]` (фиксированная)
- Ширина: `w-full` (100% колонки)
- Пропорции: не квадратная, прямоугольная

**Поля названий:**
- Каждое поле: `py-2` (padding 8px сверху и снизу)
- Высота input: ~40px
- Отступ между полями: `space-y-3` (12px)
- Общая высота: 40px + 12px + 40px + 28px (labels) = 120px

### Иконки и текст:

**Было:**
```tsx
<Upload className="w-6 h-6" />
<p className="text-gray-400 text-[10px]">...</p>
```

**Стало:**
```tsx
<Upload className="w-5 h-5" />
<p className="text-gray-400 text-[10px]">...</p>
```

### Grid структура:

```tsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-6">
  {/* Left: 1 колонка - изображение */}
  <div className="md:col-span-1">
    <div className="h-[120px]">...</div>
  </div>
  
  {/* Right: 2 колонки - названия */}
  <div className="md:col-span-2 space-y-3">
    {LOCALES.map(...)}
  </div>
</div>
```

## Примеры использования

### Модальное окно добавления:

```tsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-6">
  <div className="md:col-span-1">
    <label className="block text-xs font-medium text-gray-300 mb-1.5">
      Изображение (PNG) *
    </label>
    <div className="w-full h-[120px] bg-gray-700 rounded-lg ...">
      {/* Drag & Drop область */}
    </div>
  </div>
  
  <div className="md:col-span-2 space-y-3">
    {LOCALES.map((locale) => (
      <div key={locale.code}>
        <label className="block text-xs font-medium text-gray-300 mb-1.5">
          Название ({locale.name}) *
        </label>
        <input className="w-full px-3 py-2 ..." />
      </div>
    ))}
  </div>
</div>
```

### Модальное окно редактирования:

Точно такая же структура, как в модальном окне добавления.
