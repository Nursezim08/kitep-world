
```

Tablet:     640-1024  - Упрощенная таблица
```


## Responsive Design

### Desktop (> 1024px)
- Полная таблица со всеми колонками
- Боковая панель всегда видна
- Модальные окна по центру

### Tablet (640-1024px)
- Упрощенная таблица (скрыты некоторые колонки)
- Боковая панель сворачивается
- Модальные окна адаптируются

### Mobile (< 640px)
- Карточный вид вместо таблицы
- Боковая панель - выдвижное меню
- Модальные окна на весь экран

---

**Примечание**: Все размеры, цвета и стили соответствуют существующему дизайну админ-панели для обеспечения единообразия интерфейса.
