# Устранение проблем с навигацией

## Проблема: Кнопки навигации не работают

### Возможные причины:

1. **Два сервера запущены одновременно**
   - Проверьте, что сервер работает только на порту 3000
   - Остановите все процессы Node.js: `taskkill /F /IM node.exe`
   - Запустите сервер заново: `npm run dev`

2. **Кэш браузера**
   - Очистите кэш браузера (Ctrl+Shift+Delete)
   - Откройте страницу в режиме инкогнито
   - Жесткая перезагрузка (Ctrl+Shift+R)

3. **Неправильный порт**
   - Убедитесь, что открыта страница на `http://localhost:3000`
   - Не `http://localhost:3001` или другой порт

4. **Ошибки JavaScript**
   - Откройте консоль браузера (F12)
   - Проверьте наличие ошибок (красный текст)
   - Посмотрите, выводится ли "Navigating to: ..." при клике

### Диагностика:

1. **Откройте консоль браузера (F12)**
2. **Перейдите на http://localhost:3000/home**
3. **Кликните на кнопку навигации**
4. **Проверьте вывод в консоль:**
   - ✅ Видите "Navigating to: /catalog" → router работает, проблема в Next.js
   - ❌ Ничего не видите → клик не срабатывает, проблема в React
   - ❌ Видите ошибку → проблема в коде

### Решения:

#### Если router работает, но навигация не происходит:

```typescript
// Попробуйте использовать window.location вместо router.push
onClick={() => {
  window.location.href = item.href;
}}
```

#### Если клик не срабатывает:

```typescript
// Проверьте, что кнопка не disabled
<button
  disabled={false}
  onClick={(e) => {
    e.preventDefault();
    e.stopPropagation();
    console.log('Button clicked!');
    router.push(item.href);
  }}
>
```

#### Если есть ошибки в консоли:

- Проверьте импорты
- Проверьте, что `useRouter` импортирован из `next/navigation`
- Перезапустите сервер

### Проверка сервера:

```bash
# Остановить все процессы Node.js
taskkill /F /IM node.exe

# Очистить кэш Next.js
Remove-Item -Path ".next" -Recurse -Force

# Запустить сервер
npm run dev
```

### Проверка в браузере:

1. Откройте http://localhost:3000/home
2. Откройте консоль (F12)
3. Кликните на кнопку
4. Проверьте вывод

### Если ничего не помогает:

Используйте `<Link>` вместо `<button>`:

```typescript
import Link from 'next/link';

<Link href={item.href}>
  <div className="...">
    <item.icon />
    <span>{item.title}</span>
  </div>
</Link>
```
