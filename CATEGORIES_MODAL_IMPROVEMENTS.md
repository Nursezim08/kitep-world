# Улучшения модальных окон категорий

## Обзор изменений

Модальные окна добавления и редактирования категорий были улучшены для лучшего UX и масштабируемости.

## Изменения

### 1. Полупрозрачный фон модального окна

**Было:**
```tsx
<div className="fixed inset-0 bg-black bg-opacity-50 ...">
```

**Стало:**
```tsx
<div className="fixed inset-0 bg-black/50 backdrop-blur-sm ...">
```

**Эффект:**
- Фон теперь полупрозрачный (50% прозрачности)
- Добавлен эффект размытия (`backdrop-blur-sm`)
- Улучшена визуальная иерархия

### 2. Полноширинная область загрузки изображения

**Было:**
- Превью изображения: 128x128px (слева)
- Кнопка загрузки: справа от превью

**Стало:**
- Превью изображения: полная ширина, aspect-ratio 16:9
- Кнопка загрузки: под превью, на всю ширину
- Подсказка: центрирована под кнопкой

**Преимущества:**
- Лучшая видимость изображения
- Более удобный интерфейс
- Изображение отображается в правильных пропорциях (`object-contain`)

### 3. Динамические поля переводов

**Было:**
- Жестко заданные поля для `ru` и `kg`
- Дублирование кода для каждого языка

**Стало:**
- Динамическая генерация полей на основе массива `LOCALES`
- Автоматическое добавление полей при расширении enum `Locale`

**Структура LOCALES:**
```typescript
const LOCALES = [
  { code: 'ru', name: 'Русский', placeholder: 'Например: Канцелярия' },
  { code: 'kg', name: 'Кыргызча', placeholder: 'Мисалы: Канцелярия' },
] as const;
```

**Рендеринг:**
```tsx
{LOCALES.map((locale) => (
  <div key={locale.code}>
    <label>Название ({locale.name}) *</label>
    <input
      value={formData.translations[locale.code]}
      onChange={(e) => handleTranslationChange(locale.code, e.target.value)}
      placeholder={locale.placeholder}
    />
  </div>
))}
```

## Масштабируемость

### Добавление нового языка

Чтобы добавить новый язык (например, английский):

1. **Обновите enum в schema.prisma:**
```prisma
enum Locale {
  ru
  kg
  en
}
```

2. **Добавьте язык в LOCALES:**
```typescript
const LOCALES = [
  { code: 'ru', name: 'Русский', placeholder: 'Например: Канцелярия' },
  { code: 'kg', name: 'Кыргызча', placeholder: 'Мисалы: Канцелярия' },
  { code: 'en', name: 'English', placeholder: 'Example: Stationery' },
] as const;
```

3. **Запустите миграцию:**
```bash
npx prisma migrate dev
```

Поля для нового языка автоматически появятся в модальных окнах!

## Валидация

Валидация теперь динамическая:

```typescript
for (const locale of LOCALES) {
  if (!formData.translations[locale.code]?.trim()) {
    setError(`Введите название на ${locale.name.toLowerCase()}`);
    return;
  }
}
```

Проверяются все языки из массива `LOCALES`.
