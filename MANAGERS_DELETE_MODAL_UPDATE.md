# 🗑️ Обновление: Стильное модальное окно подтверждения удаления

## ✨ Что изменено

Заменено стандартное уведомление `confirm()` на красивое модальное окно подтверждения удаления менеджера в едином стиле с админ-панелью.

## 🎨 Новый дизайн

### Было (стандартное)
```javascript
if (!confirm('Вы уверены, что хотите удалить этого менеджера?')) return;
```
- Стандартное системное окно браузера
- Не соответствует дизайну админ-панели
- Минимум информации

### Стало (стильное модальное окно)
- Красивое модальное окно в стиле админ-панели
- Темная тема с фиолетовыми акцентами
- Подробная информация о менеджере
- Предупреждение о последствиях
- Анимации и переходы

## 🔧 Технические изменения

### 1. Добавлены новые состояния
```typescript
const [showDeleteModal, setShowDeleteModal] = useState(false);
const [managerToDelete, setManagerToDelete] = useState<Manager | null>(null);
```

### 2. Обновлена функция удаления
```typescript
// Новая функция для открытия модального окна
const openDeleteModal = (manager: Manager) => {
  setManagerToDelete(manager);
  setShowDeleteModal(true);
};

// Обновленная функция удаления
const handleDeleteManager = async (managerId: string) => {
  // Удален confirm(), добавлено закрытие модального окна
  setShowDeleteModal(false);
  setManagerToDelete(null);
  // ... остальная логика
};
```

### 3. Обновлен вызов в кнопке
```typescript
// Было
onClick={() => handleDeleteManager(manager.id)}

// Стало
onClick={() => openDeleteModal(manager)}
```

## 🔄 Поведение

### Открытие модального окна
1. Пользователь нажимает иконку удаления 🗑️
2. Открывается модальное окно с информацией о менеджере
3. Показывается предупреждение о последствиях

### Подтверждение удаления
1. Пользователь нажимает "Удалить" в модальном окне
2. Выполняется API запрос на удаление
3. Модальное окно закрывается
4. Список менеджеров обновляется

### Отмена
1. Пользователь нажимает "Отмена" или ✕
2. Модальное окно закрывается
3. Никаких изменений не происходит
