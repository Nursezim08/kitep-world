# Финальная версия модальных окон категорий

## Обзор изменений

Модальные окна добавления и редактирования категорий были полностью переработаны для улучшения UX.

## Ключевые изменения

### 1. Убран черный фон ❌

**Было:**
```tsx
<div className="fixed inset-0 bg-black/50 backdrop-blur-sm ...">
```

**Стало:**
```tsx
<div className="fixed inset-0 flex items-center justify-center z-50 p-4">
```

**Результат:**
- Полностью прозрачный фон
- Модальное окно "парит" над страницей
- Добавлена тень для глубины (`shadow-2xl`)
- Фокус полностью на модальном окне

### 2. Уменьшена область загрузки изображения 📐

**Было:**
- Высота: `aspect-video` (16:9, ~360px для ширины 640px)

**Стало:**
- Высота: фиксированная `h-48` (192px)
- Более компактное превью
- Меньше прокрутки в модальном окне

**Визуальное сравнение:**

```
БЫЛО (aspect-video):
┌─────────────────────────────────┐
│                                 │
│                                 │
│         [Изображение]           │
│                                 │
│                                 │
└─────────────────────────────────┘
Высота: ~360px

СТАЛО (h-48):
┌─────────────────────────────────┐
│         [Изображение]           │
└─────────────────────────────────┘
Высота: 192px
```

### 3. Статус как переключатель (Toggle Switch) 🔘

**Было:**
```tsx
<select>
  <option value="active">Активна</option>
  <option value="inactive">Неактивна</option>
</select>
```

**Стало:**
```tsx
<button
  type="button"
  onClick={() => toggleStatus()}
  className={`relative inline-flex h-8 w-16 items-center rounded-full ${
    status === 'active' ? 'bg-blue-600' : 'bg-gray-600'
  }`}
>
  <span className={`inline-block h-6 w-6 transform rounded-full bg-white ${
    status === 'active' ? 'translate-x-9' : 'translate-x-1'
  }`} />
</button>
<span>Активна / Неактивна</span>
```

**Визуализация:**

```
Активна:
[●────────] Активна
 ↑ синий фон

Неактивна:
[────────●] Неактивна
         ↑ серый фон
```

**Преимущества:**
- ✅ Интуитивно понятный интерфейс
- ✅ Визуальная обратная связь
- ✅ Современный дизайн
- ✅ Меньше кликов (один клик вместо двух)
- ✅ Анимированный переход

## Технические детали

### Toggle Switch компонент

```tsx
<button
  type="button"
  onClick={() => setFormData({ 
    ...formData, 
    status: formData.status === 'active' ? 'inactive' : 'active' 
  })}
  className={`relative inline-flex h-8 w-16 items-center rounded-full 
    transition-colors focus:outline-none focus:ring-2 focus:ring-blue-500 
    focus:ring-offset-2 focus:ring-offset-gray-800 ${
    formData.status === 'active' ? 'bg-blue-600' : 'bg-gray-600'
  }`}
>
  <span
    className={`inline-block h-6 w-6 transform rounded-full bg-white 
      transition-transform ${
      formData.status === 'active' ? 'translate-x-9' : 'translate-x-1'
    }`}
  />
</button>
<span className="ml-3 text-sm text-gray-400">
  {formData.status === 'active' ? 'Активна' : 'Неактивна'}
</span>
```

### Размеры и отступы

```css
/* Превью изображения */
height: 12rem (192px)
width: 100%
object-fit: contain

/* Toggle switch */
height: 2rem (32px)
width: 4rem (64px)

/* Кнопка внутри toggle */
height: 1.5rem (24px)
width: 1.5rem (24px)

/* Позиции кнопки */
Неактивна: translate-x-1 (4px от левого края)
Активна: translate-x-9 (36px от левого края)
```

### Анимации

```css
/* Toggle switch */
transition-colors: плавное изменение цвета фона
transition-transform: плавное перемещение кнопки

/* Focus ring */
focus:ring-2 focus:ring-blue-500
focus:ring-offset-2 focus:ring-offset-gray-800
```

## Примеры использования

### Создание активной категории

1. Открыть модальное окно
2. Загрузить изображение
3. Заполнить названия
4. Toggle switch уже в положении "Активна" (по умолчанию)
5. Нажать "Создать"

### Создание неактивной категории

1. Открыть модальное окно
2. Загрузить изображение
3. Заполнить названия
4. Кликнуть на toggle switch → переключится в "Неактивна"
5. Нажать "Создать"

### Изменение статуса при редактировании

1. Открыть категорию для редактирования
2. Toggle switch показывает текущий статус
3. Кликнуть для переключения
4. Нажать "Сохранить"
