# Настройка юридических страниц

## Обзор

В проекте реализованы две юридические страницы в соответствии с законодательством Кыргызской Республики:
- **Политика конфиденциальности** (`/privacy`)
- **Условия использования** (`/terms`)

## Что нужно заполнить

### 1. Информация о компании

В обеих страницах замените следующие плейсхолдеры на реальные данные:

#### В файле `app/(legal)/privacy/page.tsx`:

```tsx
// Строка ~30
Оператором персональных данных является: <strong>[Название юридического лица]</strong>, 
зарегистрированное по адресу: <strong>[Юридический адрес]</strong>, 
ИНН: <strong>[ИНН]</strong>.
```

#### В файле `app/(legal)/terms/page.tsx`:

```tsx
// Строка ~30
Оператором Сервиса является: <strong>[Название юридического лица]</strong>, 
зарегистрированное в соответствии с законодательством Кыргызской Республики, 
ИНН: <strong>[ИНН]</strong>.
```

### 2. Контактная информация

#### Политика конфиденциальности (privacy/page.tsx):

```tsx
// Строка ~400+
<p><strong>Email:</strong> privacy@nur-kitep.kg</p>
<p><strong>Телефон:</strong> +996 XXX XXX XXX</p>
<p><strong>Адрес:</strong> [Физический адрес офиса]</p>
```

#### Условия использования (terms/page.tsx):

```tsx
// Строка ~570+
<p><strong>Оператор:</strong> [Название юридического лица]</p>
<p><strong>ИНН:</strong> [ИНН]</p>
<p><strong>Юридический адрес:</strong> [Адрес]</p>
<p><strong>Email:</strong> support@nur-kitep.kg</p>
<p><strong>Телефон:</strong> +996 XXX XXX XXX</p>
```

| `[Название юридического лица]` | `ОсОО "Нур-Китеп"` |
| `[ИНН]` | `12345678901234` |
| `[Юридический адрес]` | `г. Бишкек, ул. Киевская 147` |
| `[Физический адрес офиса]` | `г. Бишкек, ул. Киевская 147, оф. 205` |

## Настройка email адресов

### 1. Создайте email адреса:
- `privacy@nur-kitep.kg` - для запросов о персональных данных
- `support@nur-kitep.kg` - для общей поддержки

### 2. Настройте переадресацию:
Вы можете переадресовать эти адреса на существующий email:
```
privacy@nur-kitep.kg → admin@nur-kitep.kg
support@nur-kitep.kg → admin@nur-kitep.kg
```

## Юридическая проверка

⚠️ **Важно:** Перед публикацией рекомендуется проверить документы с юристом!

Убедитесь что:
- ✅ Название компании соответствует регистрационным документам
- ✅ ИНН указан верно
- ✅ Юридический адрес актуален
- ✅ Контактные данные работают
- ✅ Сроки (14 дней возврат, 30 дней претензия) соответствуют вашей политике

## Законодательство КР

Документы разработаны в соответствии с:
- **Закон КР "О персональных данных"** от 14 апреля 2008 года № 58
- **Закон КР "О защите прав потребителей"**
- **Гражданский кодекс КР**

## Проверка после заполнения

1. Откройте `/privacy` - убедитесь что нет плейсхолдеров `[...]`
2. Откройте `/terms` - убедитесь что нет плейсхолдеров `[...]`
3. Проверьте корректность всех email адресов
4. Проверьте корректность телефонов
5. Убедитесь что ссылки на документы работают

## Обновление дат

Дата последнего обновления обновляется автоматически через JavaScript:
```tsx
{new Date().toLocaleDateString('ru-RU')}
```

При существенных изменениях в документах обязательно уведомите пользователей!
