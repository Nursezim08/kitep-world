# Сводка: Система входа для менеджеров

## ✅ Реализовано

### Страницы (3)

1. **`/manager/login`** - Страница входа
   - Форма с email и паролем
   - Синий дизайн (отличается от админской панели)
   - Валидация полей
   - Обработка ошибок

2. **`/manager/verify`** - Страница подтверждения
   - Ввод 6-значного кода
   - Кнопка повторной отправки
   - Автоматический переход после успешной проверки

3. **`/manager/branch/[id]`** - Панель филиала
   - Информация о филиале
   - Список менеджеров
   - Кнопка выхода

### API Endpoints (3)

1. **`POST /api/manager/login`**
   - Проверка учетных данных
   - Проверка роли и привязки к филиалу
   - Генерация и отправка кода на email

2. **`POST /api/manager/verify-code`**
   - Проверка кода
   - Создание JWT токена
   - Установка cookie

3. **`POST /api/manager/resend-code`**
   - Повторная отправка кода
   - Удаление старых кодов

### Компоненты

1. **`BranchManagerClient.tsx`**
   - Client component для панели филиала
   - Отображение информации о филиале
   - Список менеджеров
   - Функция выхода

### Утилиты

1. **`lib/email.ts`**
   - Функция `sendManagerLoginEmail()`
   - HTML шаблон с синим дизайном
   - Поддержка dev режима (вывод в консоль)

### База данных

- Использует существующую таблицу `email_verifications`
- Совместима с текущей схемой
- Не требует миграций

## 🔄 Процесс входа

```
1. Менеджер → /manager/login
   ↓
2. Ввод email + пароль
   ↓
3. POST /api/manager/login
   ↓
4. Проверки:
   - Учетные данные ✓
   - Роль manager ✓
   - Привязка к филиалу ✓
   - Статус active ✓
   ↓
5. Генерация кода (6 цифр)
   ↓
6. Отправка на email
   ↓
7. Менеджер → /manager/verify
   ↓
8. Ввод кода
   ↓
9. POST /api/manager/verify-code
   ↓
10. Проверка кода
    ↓
11. Создание JWT токена
    ↓
12. Установка cookie
    ↓
13. Redirect → /manager/branch/[id]
    ↓
14. Панель филиала ✓
```

## 🎯 Использование

### Для администратора:
1. Создайте менеджера в `/admin/managers`
2. Назначьте его на филиал в `/admin/branches`
3. Передайте менеджеру email и пароль

### Для менеджера:
1. Откройте `/manager/login`
2. Введите email и пароль
3. Проверьте email и введите код
4. Управляйте филиалом
