### Исправление: Белая страница при клике на товар (06.05.2026)

**Проблема:**
- При клике на карточку товара открывалась белая страница
- Страница детального просмотра товара не загружалась
- Причина: в Next.js 15 параметр `params` является Promise

**Решение:**
- ✅ Обновлен серверный компонент `app/admin/products/[id]/page.tsx`
- ✅ Изменен тип `params` с `{ id: string }` на `Promise<{ id: string }>`
- ✅ Добавлен `await params` перед использованием
- ✅ Обновлены все методы API endpoint `/api/admin/products/[id]/route.ts`
- ✅ Улучшена обработка ошибок в `ProductDetailClient`

**Изменения:**

*Серверный компонент:*
```typescript
// Было:
{ params }: { params: { id: string } }
return <ProductDetailClient user={user} productId={params.id} />;

// Стало:
{ params }: { params: Promise<{ id: string }> }
const { id } = await params;
return <ProductDetailClient user={user} productId={id} />;
```

*API endpoints:*
```typescript
// Было:
```

### Исправление автоматического открытия выпадающего списка районов (06.05.2026)

**Исправлена проблема с автоматическим открытием выпадающего списка:**

**Проблема:**
- При открытии модального окна редактирования филиала автоматически открывался выпадающий список районов
- Это происходило из-за `useEffect`, который загружал районы при изменении города
- Пользователь видел открытый список, хотя не кликал на поле

**Решение:**
- Добавлен параметр `shouldOpen` в функцию `searchDistricts`
- При загрузке районов через `useEffect` передается `shouldOpen = false`
- Список открывается только при явном взаимодействии пользователя (фокус, ввод)

**Реализация:**

```typescript
// Поиск районов с параметром shouldOpen
const searchDistricts = async (query: string, shouldOpen: boolean = true) => {
  // ... код загрузки
  if (response.ok && data.districts) {
    setDistricts(data.districts);
    // Открываем список только если shouldOpen = true
    if (shouldOpen) {
      setIsOpen(data.districts.length > 0);
    }
  }
};

// Загрузка районов при изменении города (без автоматического открытия)
useEffect(() => {
  if (city) {
    searchDistricts('', false); // shouldOpen = false
  }
}, [city]);
```

**Поведение:**
- ✅ При изменении города районы загружаются в фоне
- ✅ Список НЕ открывается автоматически
- ✅ Список открывается только при фокусе на поле или вводе текста
- ✅ Улучшенный UX без отвлекающих элементов

```typescript
const editModalRef = useRef<HTMLDivElement>(null);
const editModalContainerRef = useRef<HTMLDivElement>(null);

// Автоматический скролл вверх при открытии модального окна
useEffect(() => {
  if (showEditModal && editModalContainerRef.current) {
    setTimeout(() => {
      if (editModalContainerRef.current) {
        editModalContainerRef.current.scrollTop = 0;
      }
    }, 50);
  }

// В JSX:
<div 
  ref={editModalContainerRef}
  className="fixed inset-0 ... flex items-start ... overflow-y-auto"
>
  <div ref={editModalRef} className="bg-[#252d3d] ...">
    {/* Контент модального окна */}
  </div>
</div>
```

### Автоматический скролл вверх при открытии модальных окон филиалов (06.05.2026)

**Исправлена проблема с отсутствием скролла при открытии модальных окон:**

**Проблема:**
- При открытии модального окна редактирования филиала верхняя часть (название и код) не была видна
- Модальное окно имеет внутренний скролл, а не скролл всей страницы
- Пользователь видел середину формы вместо начала

**Решение:**
- Добавлен `useRef` для контейнера модального окна
- При открытии модального окна скроллится его контейнер, а не вся страница
- Используется `scrollTop = 0` для мгновенного скролла к началу

**Реализация:**

*Для модального окна редактирования (встроенное):*
```typescript
const editModalRef = useRef<HTMLDivElement>(null);

const openEditModal = (branch: Branch) => {
  // ... установка данных
  setShowEditModal(true);
  
  setTimeout(() => {
    if (editModalRef.current) {
      editModalRef.current.scrollTop = 0;
    }
  }, 100);
};

// В JSX:
<div ref={editModalRef} className="fixed inset-0 ... overflow-y-auto">
```

*Для модального окна добавления (компонент):*
```typescript
const modalRef = useRef<HTMLDivElement>(null);

useEffect(() => {
  if (isOpen && modalRef.current) {
    modalRef.current.scrollTop = 0;
  }
}, [isOpen]);

// В JSX:
<div ref={modalRef} className="fixed inset-0 ... overflow-y-auto">
```

**Файлы:**
- `app/admin/branches/BranchesClient.tsx` - Скролл для модального окна редактирования
- `app/admin/branches/AddBranchModal.tsx` - Скролл для модального окна добавления

**Реализация:**
```typescript
const img = new Image();
img.onload = () => {
  if (img.width !== img.height) {
    setError('Изображение должно быть квадратным (например: 512x512, 1024x1024)');
    return;
  }
  // Сохраняем только если квадратное
  setFormData({ ...formData, image: base64 });
};
img.src = base64;
```

**Валидация (полная):**
1. ✅ Формат: только PNG

**Допустимые размеры:**

**Недопустимые размеры:**

**Сообщение об ошибке:**
