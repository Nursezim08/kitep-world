# Drag and Drop для загрузки изображений категорий

## Обзор

Реализован современный интерфейс загрузки изображений с поддержкой drag and drop (перетаскивания файлов).

## Функционал

### 1. Drag and Drop (Перетаскивание)
- Перетащите PNG файл на область загрузки
- Визуальная обратная связь при наведении
- Автоматическая валидация формата и размера

### 2. Click to Upload (Клик для загрузки)
- Клик по области открывает диалог выбора файла
- Альтернативный способ загрузки для пользователей

### 3. Визуальные состояния
- **Пустое:** Показывает инструкции и иконку
- **Наведение (Dragging):** Синяя подсветка
- **С изображением:** Показывает превью

## Технические детали

### Обработчики событий

```typescript
// Предотвращение поведения по умолчанию
const handleDragOver = (e: React.DragEvent<HTMLDivElement>) => {
  e.preventDefault();
  e.stopPropagation();
  setIsDragging(true);
};

// Сброс состояния при уходе курсора
const handleDragLeave = (e: React.DragEvent<HTMLDivElement>) => {
  e.preventDefault();
  e.stopPropagation();
  setIsDragging(false);
};

// Обработка сброшенного файла
const handleDrop = (e: React.DragEvent<HTMLDivElement>) => {
  e.preventDefault();
  e.stopPropagation();
  setIsDragging(false);

  const file = e.dataTransfer.files?.[0];
  if (file) {
    processImageFile(file);
  }
};

// Клик для открытия диалога
const handleClick = () => {
  document.getElementById('image-upload-input')?.click();
};
```

### Обработка файла

```typescript
const processImageFile = (file: File) => {
  // Проверка формата
  if (!file.type.startsWith('image/png')) {
    setError('Допускаются только PNG изображения');
    return;
  }

  // Проверка размера (макс 5MB)
  if (file.size > 5 * 1024 * 1024) {
    setError('Размер изображения не должен превышать 5MB');
    return;
  }

  // Чтение файла
  const reader = new FileReader();
  reader.onloadend = () => {
    const base64 = reader.result as string;
    setFormData({ ...formData, image: base64 });
    setImagePreview(base64);
    setError('');
  };
  reader.readAsDataURL(file);
};
```

### Стили

```tsx
<div

>
```

## События и их обработка

### Последовательность событий при drag & drop

1. **dragenter** → Файл входит в зону
2. **dragover** → Файл над зоной (постоянно)
3. **dragleave** → Файл покидает зону
4. **drop** → Файл отпущен

```typescript
}

// drop - обрабатываем файл

}
```

## Валидация квадратных изображений

### Требование

Загружаемое изображение должно быть квадратным (ширина = высота).

### Реализация

```typescript
const processImageFile = (file: File) => {
  // 1. Проверка формата
  if (!file.type.startsWith('image/png')) {
    setError('Допускаются только PNG изображения');
    return;
  }

  // 2. Проверка размера
  if (file.size > 5 * 1024 * 1024) {
    setError('Размер изображения не должен превышать 5MB');
    return;
  }

  // 3. Проверка пропорций (квадратное)
  const img = new Image();
  const reader = new FileReader();
  
  reader.onloadend = () => {
    const base64 = reader.result as string;
    
    img.onload = () => {
      if (img.width !== img.height) {
        setError('Изображение должно быть квадратным (например: 512x512, 1024x1024)');
        return;
      }
      
      // Всё ок, сохраняем
      setFormData({ ...formData, image: base64 });
      setImagePreview(base64);
      setError('');
    };
    
    img.src = base64;
  };
  
  reader.readAsDataURL(file);
};
```

### Логика проверки

1. **Чтение файла:** FileReader преобразует файл в base64
2. **Создание Image:** Создаётся объект Image для получения размеров
3. **Проверка пропорций:** Сравнивается `img.width` и `img.height`
4. **Результат:**
   - Если квадратное → сохраняется
   - Если не квадратное → показывается ошибка

### Примеры

#### ✅ Допустимые размеры
- 256x256
- 512x512
- 1024x1024
- 2048x2048
- 4096x4096

#### ❌ Недопустимые размеры
- 512x256 (прямоугольник)
- 1024x768 (прямоугольник)
- 1920x1080 (прямоугольник)

### Сообщение об ошибке

```
❌ Изображение должно быть квадратным (например: 512x512, 1024x1024)
```

### Обновлённая подсказка

**Было:**
```

```

**Стало:**
```

```

### Полная валидация

| Проверка | Условие | Сообщение об ошибке |
|----------|---------|---------------------|
| Формат | PNG | Допускаются только PNG изображения |
| Размер | ≤ 5MB | Размер изображения не должен превышать 5MB |
| Пропорции | width = height | Изображение должно быть квадратным |


### Технические детали

#### Асинхронная проверка

Проверка пропорций асинхронная, так как требует:
1. Чтения файла (FileReader)
2. Загрузки изображения (Image.onload)

#### Порядок событий

```typescript

  ↓
FileReader.onloadend
  ↓
Image.src = base64
  ↓
Image.onload
  ↓
Проверка width === height
  ↓
Сохранение или ошибка
```
