
```typescript

}
```

## 🔄 Процесс работы

### Создание категории

```
1. Пользователь загружает PNG изображение
   ↓
2. Клиент конвертирует в base64
   ↓
3. Отправка на сервер (POST /api/admin/categories)
   ↓
4. Сервер проверяет: isBase64Image(image)
   ↓
5. Загрузка в S3: uploadImageToS3(image, 'categories')
   ↓
6. Получение URL: https://s3.twcstorage.ru/nur-kitep/categories/uuid.png
   ↓
7. Сохранение URL в БД
   ↓
8. Возврат категории с URL
```

### Обновление категории

```
1. Пользователь загружает новое изображение
   ↓
2. Клиент конвертирует в base64
   ↓
3. Отправка на сервер (PATCH /api/admin/categories/[id])
   ↓
4. Сервер проверяет: isBase64Image(newImage)
   ↓
5. Загрузка нового в S3: uploadImageToS3(newImage, 'categories')
   ↓
6. Проверка старого: isS3Image(oldImage)
   ↓
7. Удаление старого из S3: deleteImageFromS3(oldImage)
   ↓
8. Обновление URL в БД
   ↓
9. Возврат обновлённой категории
```

### Удаление категории

```
1. Пользователь удаляет категорию
   ↓
2. Отправка на сервер (DELETE /api/admin/categories/[id])
   ↓
3. Проверка: нет подкатегорий и товаров
   ↓
4. Получение категории с image URL
   ↓
5. Мягкое удаление: status = 'deleted'
   ↓
6. Проверка: isS3Image(image)
   ↓
7. Удаление из S3: deleteImageFromS3(image)
   ↓
8. Возврат успеха
```
