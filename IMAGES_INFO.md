## Параметры URL

### Размер изображения
- `w=800` - ширина 800px
- `h=800` - высота 800px

### Обрезка
- `fit=crop` - обрезка по центру для квадратного формата

### Примеры использования

**Товар:**
```
https://images.unsplash.com/photo-ID?w=800&h=800&fit=crop
```

**Категория:**
```
https://images.unsplash.com/photo-ID?w=512&h=512&fit=crop
```

**Большое изображение:**
```
https://images.unsplash.com/photo-ID?w=1920&h=1080&fit=crop
```

## Замена изображений

### Вариант 1: Через seed скрипт

Откройте `prisma/seed.ts` и измените URL:

```typescript
const categoryImages: Record<string, string[]> = {
  'Канцелярия': [
    'https://your-new-image-url-1.jpg',
    'https://your-new-image-url-2.jpg',
    'https://your-new-image-url-3.jpg',
    'https://your-new-image-url-4.jpg',
  ],
  // ... остальные категории
};
```

Затем запустите:
```bash
npm run db:seed
```

### Вариант 2: Через админку

1. Откройте товар в админке
2. Нажмите "Редактировать"
3. Загрузите новые изображения
4. Сохраните

### Вариант 3: Загрузка в S3

1. Загрузите изображения в S3 bucket
2. Получите публичные URL
3. Обновите через seed или админку

## Оптимизация

### Текущие параметры:
- ✅ Квадратный формат (800x800px)
- ✅ Обрезка по центру (fit=crop)
- ✅ Оптимизированный JPEG
- ✅ CDN доставка

### Дополнительные параметры Unsplash:

**Качество:**
```
?w=800&h=800&fit=crop&q=80
```
(q=80 - качество 80%)

**Формат:**
```
?w=800&h=800&fit=crop&fm=webp
```
(fm=webp - формат WebP для меньшего размера)

**Автоматическая оптимизация:**
```
?w=800&h=800&fit=crop&auto=format,compress
```

## Troubleshooting

### Изображения не загружаются

**Причина 1:** Unsplash недоступен
- Проверьте интернет-соединение
- Попробуйте открыть URL в браузере

**Причина 2:** Блокировка CORS
- Unsplash поддерживает CORS
- Проверьте настройки браузера

**Причина 3:** Неверный URL
- Проверьте формат URL
- Убедитесь, что photo-ID корректный

### Медленная загрузка

**Решение 1:** Уменьшите размер
```
?w=400&h=400&fit=crop
```

**Решение 2:** Используйте WebP
```
?w=800&h=800&fit=crop&fm=webp
```

**Решение 3:** Добавьте качество
```
?w=800&h=800&fit=crop&q=75
```
