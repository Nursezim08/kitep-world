## Следующие шаги

### 1. Убедитесь, что бот запущен

1. Откройте Telegram
2. Найдите вашего бота (по имени из @BotFather)
3. Нажмите кнопку **Start** или отправьте `/start`

### 2. Попробуйте войти

1. Откройте `http://localhost:3000/admin/login`
2. Введите:
   - Email: `nursezim416@gmail.com`
   - Пароль: ваш пароль
3. Нажмите "Войти"

### 3. Проверьте Telegram

Вы должны получить сообщение:
```
🔐 Код подтверждения входа в админ-панель

👤 Пользователь: Нурсезим
📧 Email: nursezim416@gmail.com

🔢 Ваш код: 123456

⏱ Код действителен 5 минут.
```

### 4. Введите код

1. Скопируйте код из Telegram
2. Вставьте на странице `/admin/verify`
3. Нажмите "Подтвердить"

### 5. Готово! 🎉

Вы будете перенаправлены в админ-панель `/admin/dashboard`

## Если код не приходит

### Проверьте логи сервера

В консоли Next.js должны быть логи:
```
[Admin Login] Login attempt for email: nursezim416@gmail.com
[Telegram] Sending message to chat_id: 1434922404
[Telegram] SUCCESS: Message sent successfully
```

### Проверьте настройки

```bash
node scripts/diagnose-admin.js
```

### Проверьте бота

1. Бот должен быть запущен (Start)
2. Bot Token должен быть правильным
3. Telegram User ID должен быть правильным

### Проверьте Telegram User ID

Чтобы узнать свой Telegram User ID:
1. Откройте [@userinfobot](https://t.me/userinfobot)
2. Отправьте любое сообщение
3. Бот вернет ваш User ID
4. Сравните с `ADMIN_TELEGRAM_USER_ID` в базе данных
