# 🚀 Быстрый старт с seed данными

## Одна команда для заполнения БД

```bash
npm run db:seed
```

## Что получите

✅ **10 категорий** с изображениями и переводами  
✅ **100 товаров** (по 10 в каждой категории)  
✅ **400 изображений** товаров  
✅ **Переводы** на русском и кыргызском  

## Категории

1. 📎 **Канцелярия** - степлеры, дыроколы, ножницы (10 товаров)
2. 📚 **Книги** - классика, учебники, энциклопедии (10 товаров)
3. 📓 **Тетради** - школьные, блокноты, дневники (10 товаров)
4. ✒️ **Ручки** - шариковые, гелевые, автоматические (10 товаров)
5. ✏️ **Карандаши** - простые, цветные, механические (10 товаров)
6. 📁 **Папки и файлы** - скоросшиватели, файлы (10 товаров)
7. 🎨 **Краски и кисти** - акварель, гуашь, кисти (10 товаров)
8. 🎒 **Рюкзаки** - школьные, спортивные (10 товаров)
9. 📦 **Пеналы** - школьные, силиконовые (10 товаров)
10. 🖨️ **Офисная техника** - калькуляторы, ламинаторы (10 товаров)

## Полезные команды

```bash
# Заполнить БД
npm run db:seed

# Просмотр данных
npx prisma studio

# Сброс БД (удалит все данные!)
npx prisma migrate reset

# Dev сервер
npm run dev
```

## ⚠️ Важно

Команда `npm run db:seed` **удаляет все существующие** категории и товары перед созданием новых!
