# Кастомный компонент Select

## Описание

Создан кастомный компонент выпадающего списка `CustomSelect`, который заменяет стандартные HTML `<select>` элементы и полностью соответствует дизайн-системе админ-панели.

## Файл компонента

**Путь**: `app/components/CustomSelect.tsx`

## Особенности

### 🎨 Дизайн
- **Темная тема**: Фон `#1e2533`, при ховере `#2a3347`
- **Фиолетовый акцент**: Выбранный элемент подсвечивается фиолетовым
- **Иконки**: ChevronDown для индикации состояния, Check для выбранного элемента
- **Анимации**: Плавное открытие/закрытие, вращение стрелки
- **Границы**: Тонкие границы `gray-700/50`
- **Скругление**: `rounded-lg` для мягкого вида

### ⚡ Функциональность
- **Клик вне области**: Автоматически закрывает выпадающий список
- **Keyboard-friendly**: Можно использовать с клавиатуры
- **Скролл**: Автоматический скролл для длинных списков (max-height: 240px)
- **Визуальная обратная связь**: Hover эффекты, выделение выбранного элемента
- **Focus состояние**: Фиолетовое кольцо при фокусе

### 🔧 Технические детали
- **TypeScript**: Полная типизация
- **React Hooks**: useState, useRef, useEffect
- **Управление состоянием**: Контролируемый компонент
- **Обработка событий**: Click outside detection
- **Анимации**: Tailwind CSS animations

## API компонента

### Props

```typescript
interface CustomSelectProps {
  value: string;              // Текущее выбранное значение
  onChange: (value: string) => void;  // Callback при изменении
  options: Option[];          // Массив опций
  placeholder?: string;       // Плейсхолдер (по умолчанию: "Выберите...")
  className?: string;         // Дополнительные CSS классы
}

interface Option {
  value: string;              // Значение опции
  label: string;              // Отображаемый текст
}
```

### Пример использования

```tsx
import CustomSelect from '@/app/components/CustomSelect';

// В компоненте
<CustomSelect
  value={selectedValue}
  onChange={setSelectedValue}
  options={[
    { value: 'option1', label: 'Опция 1' },
    { value: 'option2', label: 'Опция 2' },
    { value: 'option3', label: 'Опция 3' },
  ]}
  placeholder="Выберите опцию"
/>
```

- **Фильтр по филиалу**: Все филиалы / Не назначен / Список филиалов

- **Выбор статуса**: Активен / Неактивен / Заблокирован

### ✅ Функциональность
- Программное управление состоянием

## Структура компонента

```
CustomSelect
├── Select Button (кнопка с текущим значением)
│   ├── Текст выбранной опции или placeholder
│   └── Иконка ChevronDown (вращается при открытии)
│
└── Dropdown Menu (выпадающее меню)
    └── Список опций
        ├── Опция 1 (с галочкой если выбрана)
        ├── Опция 2
        └── Опция N
```

## Стили

### Кнопка
- Фон: `#1e2533`
- Hover: `#2a3347`
- Граница: `gray-700/50`
- Текст: белый / серый (placeholder)
- Padding: `px-3 py-2`
- Скругление: `rounded-lg`

### Выпадающее меню
- Фон: `#252d3d`
- Граница: `gray-700/50`
- Тень: `shadow-xl`
- Max высота: `240px` (15rem)
- Скругление: `rounded-lg`

### Опции
- Обычная: серый текст, прозрачный фон
- Hover: белый текст, фон `#2a3347`
- Выбранная: фиолетовый текст, фон `violet-500/20`, галочка

## Анимации

### Открытие/закрытие
```css
animate-in fade-in slide-in-from-top-2 duration-200
```
- Плавное появление (fade-in)
- Слайд сверху (slide-in-from-top-2)
- Длительность: 200ms

### Вращение стрелки
```css
transition-transform duration-200
rotate-180 (когда открыто)
```

## Accessibility (будущие улучшения)

Можно добавить:
- `role="listbox"` на выпадающее меню
- `role="option"` на опции
- `aria-expanded` на кнопку
- `aria-selected` на выбранную опцию
- Навигация клавиатурой (Arrow Up/Down, Enter, Escape)
- `aria-label` для screen readers

## Возможные расширения

### Поиск
Добавить поле поиска внутри выпадающего списка для фильтрации опций

### Множественный выбор
Поддержка выбора нескольких значений с чекбоксами

### Группировка
Группировка опций по категориям с заголовками

### Иконки в опциях
Добавление иконок к каждой опции для лучшей визуализации

### Виртуализация
Для очень длинных списков (1000+ элементов) использовать виртуальный скролл

## Производительность

- **Легковесный**: Минимальные зависимости (только lucide-react для иконок)
- **Оптимизированный**: useEffect только для click outside detection
- **Мемоизация**: При необходимости можно добавить useMemo для опций
- **Ленивый рендеринг**: Выпадающее меню рендерится только когда открыто

## Пример интеграции

```tsx
// Импорт
import CustomSelect from '@/app/components/CustomSelect';

// Состояние
const [status, setStatus] = useState('all');

// Использование
<div>
  <label className="block text-xs font-semibold text-gray-400 mb-2">
    Статус
  </label>
  <CustomSelect
    value={status}
    onChange={setStatus}
    options={[
      { value: 'all', label: 'Все статусы' },
      { value: 'active', label: 'Активен' },
      { value: 'inactive', label: 'Неактивен' },
    ]}
  />
</div>
```
