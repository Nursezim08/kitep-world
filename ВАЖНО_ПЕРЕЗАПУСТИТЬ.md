## Как перезапустить сервер

### Вариант 1: Через терминал

1. Остановите текущий сервер: `Ctrl + C`
2. Запустите снова: `npm run dev`

### Вариант 2: Через PowerShell (если сервер завис)

```powershell
# Остановить все процессы Node.js
Get-Process node -ErrorAction SilentlyContinue | Stop-Process -Force

# Запустить сервер
npm run dev
```

## После перезапуска

1. Откройте `http://localhost:3000/admin/login`
2. Введите credentials
3. Проверьте, что код приходит в Telegram
4. Войдите в админку

## Проверка

После перезапуска в логах не должно быть ошибки:
```
❌ The column `users.telegram_id` does not exist
```

Должны быть успешные логи:
```
✅ [Admin Login] Login attempt for email: ...
✅ [Telegram] SUCCESS: Message sent successfully
```
