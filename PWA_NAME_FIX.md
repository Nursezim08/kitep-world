# Исправление названия PWA (13.06.2026)

## Изменение

Обновлено название приложения при установке с **"Nur-Kitep"** на **"Nur-kitep"** (с маленькой буквы "k").

## Обновлённые файлы

### 1. `public/manifest.json`
```json
{
  "name": "Nur-kitep - Канцелярский магазин",
  "short_name": "Nur-kitep",
  // ...
}
```

**Было:**
- `"name": "Nur-Kitep - Канцелярский магазин"`
- `"short_name": "Nur-Kitep"`

**Стало:**
- `"name": "Nur-kitep - Канцелярский магазин"`
- `"short_name": "Nur-kitep"`

---

### 2. `app/layout.tsx`
```typescript

  // ...
```

**Было:**

- Пользователям нужно будет **переустановить приложение**
- Или Service Worker автоматически обновит манифест при следующем посещении
- Название изменится автоматически после обновления

## Очистка старой версии (для пользователей)

Если установленное приложение всё ещё показывает старое название:

**Android:**
1. Удалите приложение с домашнего экрана
2. Очистите кеш браузера
3. Установите заново

**iOS:**
1. Удалите иконку с домашнего экрана
2. Очистите кеш Safari (Настройки → Safari → Очистить историю)
3. Добавьте заново

**Desktop:**
1. Удалите приложение через меню Chrome (chrome://apps)
2. Очистите кеш браузера
3. Установите заново
