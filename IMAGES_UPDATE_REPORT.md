## 📞 Поддержка

### Если изображения не загружаются:

1. **Проверьте интернет-соединение**
2. **Откройте URL в браузере** - проверьте доступность Unsplash
3. **Проверьте консоль браузера** - ищите ошибки CORS
4. **Используйте альтернативные источники** - Pexels, Pixabay

### Замена изображений:

**Через seed скрипт:**
1. Откройте `prisma/seed.ts`
2. Измените URL в `categoryImages`
3. Запустите `npm run db:seed`

**Через админку:**
1. Откройте товар
2. Нажмите "Редактировать"
3. Загрузите новые изображения
