Я реализовал **простое решение без настройки сервера**, которое работает полностью в Next.js:

1. ✅ `app/api/sitemap.xml/route.ts` - API endpoint для генерации sitemap

## 🎯 Что нужно сделать ВАМ

### 1️⃣ Пересоберите проект
```bash
npm run build
```

### 2️⃣ Задеплойте на сервер
Загрузите обновленные файлы на ваш хостинг (Timeweb или другой).

### 3️⃣ Перезапустите сервер
```bash
pm2 restart nur-kitep
```
или
```bash
npm run start
```

### 4️⃣ Проверьте
Откройте в браузере:
- **https://nur-kitep.store/sitemap.xml**

Должен открыться XML файл со списком страниц вашего сайта.
