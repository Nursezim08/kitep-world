## 🔄 Повторный запуск

Если нужно пересоздать данные:

```bash
npm run db:seed
```

⚠️ **Внимание:** Скрипт удаляет все существующие категории и товары!

## 📍 Где посмотреть

### В админке:
1. Откройте: `http://localhost:3000/admin/login`
2. Войдите как администратор
3. Перейдите в "Категории" или "Товары"

### В Prisma Studio:
```bash
npx prisma studio
```

### В пользовательской части:
1. Откройте: `http://localhost:3000/home`
2. Просмотрите категории и товары
3. Перейдите в каталог

## 🎯 Следующие шаги

1. **Замените placeholder изображения** на реальные фотографии товаров
