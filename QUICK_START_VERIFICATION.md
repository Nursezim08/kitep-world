## Как протестировать

### 1. Запустите сервер разработки

```bash
npm run dev
```

### 2. Откройте страницу регистрации

Перейдите на: http://localhost:3000/register

### 3. Заполните форму регистрации

- Полное имя: Тестовый Пользователь
- Email: test@example.com
- Телефон: +996555123456 (опционально)
- Пароль: Test123456
- Подтвердите пароль: Test123456

### 4. Нажмите "Зарегистрироваться"

После успешной регистрации вы будете перенаправлены на страницу `/verify-email`

### 5. Получите код верификации

**В режиме разработки** код выводится в консоль сервера (терминал где запущен `npm run dev`):

```
==================================================
📧 EMAIL VERIFICATION CODE
==================================================
To: test@example.com
Code: 123456
==================================================
```

### 6. Введите код на странице верификации

- Введите 6-значный код из консоли
- Код можно вставить (Ctrl+V) - он автоматически распределится по полям
- При заполнении всех полей код отправится автоматически

### 7. Успешная верификация

После правильного ввода кода вы будете перенаправлены в профиль (`/profile`)

## Функции страницы верификации

### Автоматические функции
- ✅ Автоматический переход между полями при вводе
- ✅ Автоматическая отправка при заполнении всех полей
- ✅ Поддержка вставки кода (Ctrl+V)
- ✅ Возврат на предыдущее поле при Backspace

### Кнопка "Отправить повторно"
- Отправляет новый код на email
- Имеет таймер 60 секунд между запросами
- Очищает поля ввода после отправки

### Валидация
- Код должен состоять из 6 цифр
- Код действителен 10 минут
- Код можно использовать только один раз

## Проверка в базе данных

### Проверить статус верификации пользователя

```sql
SELECT id, full_name, email, email_verified 
FROM users 
WHERE email = 'test@example.com';
```

### Посмотреть коды верификации

```sql
SELECT * FROM email_verifications 
WHERE email = 'test@example.com' 
ORDER BY created_at DESC;
```

## Возможные ошибки и решения

### "Неверный или истекший код верификации"
- Проверьте, что код введен правильно
- Убедитесь, что прошло менее 10 минут с момента отправки
- Запросите новый код кнопкой "Отправить повторно"

### "Подождите минуту перед повторной отправкой кода"
- Это защита от спама
- Подождите 60 секунд перед следующим запросом

### "Пользователь с таким email уже существует"
- Email уже зарегистрирован
- Используйте другой email или войдите в существующий аккаунт
