# Обновления компонента Toast

## Изменения

### 1. ✅ Убран border, добавлена тень

**Было:**
```tsx
border-2 border-violet-500 shadow-2xl
```

**Стало:**
```tsx
shadow-lg hover:shadow-xl
transition-shadow duration-200
```

**Эффект:**
- Более мягкий и современный вид
- Тень увеличивается при наведении
- Плавный переход тени (200ms)

### 2. ✅ Секундомер не сбрасывается при наведении

**Было:**
```tsx
const startTime = Date.now();
// При каждом useEffect создавался новый startTime
```

**Стало:**
```tsx
const startTimeRef = useRef(Date.now());
const elapsedRef = useRef(0);

// Сохраняем прогресс при паузе
const currentStartTime = Date.now() - elapsedRef.current;
elapsedRef.current = totalElapsed;
```

**Эффект:**
- Таймер продолжается с того же места после паузы
- Прогресс сохраняется
- Нет сброса при наведении/уходе мыши

## Технические детали

### Управление таймером

```typescript
// Refs для сохранения состояния
const startTimeRef = useRef(Date.now());
const elapsedRef = useRef(0);

useEffect(() => {
  if (isPaused) return; // Пауза при наведении
  
  // Вычисляем стартовое время с учетом паузы
  const currentStartTime = Date.now() - elapsedRef.current;
  
  const interval = setInterval(() => {
    // Общее прошедшее время
    const totalElapsed = Date.now() - currentStartTime;
    elapsedRef.current = totalElapsed; // Сохраняем прогресс
    
    // Вычисляем оставшийся процент
    const remaining = Math.max(0, 100 - (totalElapsed / duration) * 100);
    setProgress(remaining);
    
    // Закрытие при завершении
    if (remaining === 0) {
      setIsVisible(false);
      setTimeout(onClose, 300);
    }
  }, 10);
  
  return () => clearInterval(interval);
}, [isPaused, duration, onClose]);
```

## Поведение таймера

### Сценарий 1: Без наведения
```
0s ────────────────────────────> 5s
▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
100%                              0%
```

### Сценарий 2: С наведением (старое поведение - ИСПРАВЛЕНО)
```
0s ──────> 2s [ПАУЗА] ──────> 5s [СБРОС]
▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
100%        60%         100%      0%
           ↑ Сброс при уходе мыши
```

### Сценарий 3: С наведением (новое поведение - ПРАВИЛЬНО)
```
0s ──────> 2s [ПАУЗА 3s] ──────> 5s
▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
100%        60%  60%  60%        0%
           ↑ Пауза    ↑ Продолжение
```

## Тестирование

### Тест 1: Тень при наведении
1. Показать toast
2. Навести мышь
3. **Ожидается:** Тень увеличивается (shadow-lg → shadow-xl)
4. Убрать мышь
5. **Ожидается:** Тень уменьшается (shadow-xl → shadow-lg)

### Тест 2: Сохранение прогресса
1. Показать toast
2. Подождать 2 секунды (прогресс ~60%)
3. Навести мышь (пауза)
4. Подождать 3 секунды
5. **Ожидается:** Прогресс остается ~60%
6. Убрать мышь
7. **Ожидается:** Прогресс продолжается с ~60%
8. Подождать 3 секунды
9. **Ожидается:** Toast закрывается

### Тест 3: Множественные паузы
1. Показать toast
2. Навести мышь → убрать → навести → убрать
3. **Ожидается:** Прогресс продолжается корректно
4. Общее время до закрытия = 5 секунд (без учета пауз)
