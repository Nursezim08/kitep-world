# Настройка Telegram бота для админ-панели

## Шаг 1: Бот уже создан ✅

Ваш бот: [@nur_kitep_bot](https://t.me/nur_kitep_bot)
Токен уже добавлен в `.env`

## Шаг 2: Получите свой Telegram ID

1. **Откройте Telegram**
2. **Найдите бота [@userinfobot](https://t.me/userinfobot)**
3. **Нажмите "Start" или отправьте любое сообщение**
4. **Бот вернет ваш ID** (например: `123456789`)

## Шаг 3: Добавьте ID в .env

Откройте файл `.env` и замените:

```env
TELEGRAM_ADMIN_USER_ID="YOUR_TELEGRAM_ID_HERE"
```

На:

```env
TELEGRAM_ADMIN_USER_ID="ваш_id_из_userinfobot"
```

Например:
```env
TELEGRAM_ADMIN_USER_ID="123456789"
```

## Шаг 4: Перезапустите сервер

```bash
# Остановите сервер (Ctrl+C)
# Запустите снова
npm run dev
```

## Шаг 5: Проверьте работу

1. Перейдите на `/admin/login`
2. Введите email и пароль
3. Нажмите "Войти"
4. **Код придет в Telegram** от бота @nur_kitep_bot

## Формат сообщения

Вы получите сообщение в таком формате:

```
🔐 Код подтверждения входа в админ-панель

👤 Пользователь: Ваше Имя
📧 Email: your@email.com

🔢 Ваш код: 123456

⏱ Код действителен 5 минут.

Если это были не вы, проигнорируйте это сообщение.
```

## Режим разработки (DEV MODE)

Если вы **НЕ** настроите `TELEGRAM_ADMIN_USER_ID`, код будет выводиться в консоль сервера:

## Безопасность

⚠️ **ВАЖНО:**
- Никогда не публикуйте токен бота (`TELEGRAM_BOT_TOKEN`)
- Никогда не публикуйте свой Telegram ID
- Файл `.env` добавлен в `.gitignore`
- В production используйте переменные окружения сервера

## Troubleshooting

### Код не приходит в Telegram

1. Проверьте, что вы **запустили бота** (нажали Start в @nur_kitep_bot)
2. Проверьте правильность `TELEGRAM_ADMIN_USER_ID` в `.env`
3. Проверьте логи сервера на наличие ошибок
4. Убедитесь, что перезапустили сервер после изменения `.env`

### Ошибка "Chat not found"

Это означает, что вы **не запустили бота**. Откройте [@nur_kitep_bot](https://t.me/nur_kitep_bot) и нажмите "Start".

### Как получить ID другого пользователя?

Попросите пользователя:
1. Открыть [@userinfobot](https://t.me/userinfobot)
2. Отправить любое сообщение
3. Скопировать ID и отправить вам

## Дополнительные возможности

### Несколько администраторов

Если нужно отправлять коды нескольким администраторам, можно:

1. Создать Telegram группу
2. Добавить туда бота
3. Получить ID группы (начинается с `-`)
4. Использовать ID группы вместо личного ID

### Кастомизация сообщений

Отредактируйте файл `lib/telegram.ts`, функцию `sendTelegramCode()`, переменную `message`.
