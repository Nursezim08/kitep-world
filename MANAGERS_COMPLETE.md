## 🔧 Технические детали

### Архитектура
```
Frontend (React/Next.js)
    ↓
API Routes (Next.js)
    ↓
Prisma ORM
    ↓
PostgreSQL Database
```

### Основные компоненты
- **ManagersClient.tsx** - главный UI компонент
- **API Routes** - 5 endpoints для CRUD операций
- **Локализация** - файлы переводов для 2 языков
- **Тестирование** - автоматический тестовый скрипт

### База данных
Использует существующие модели:
- `User` (role: 'manager') - данные менеджеров
- `BranchUser` - связь менеджер-филиал
- `Branch` - список филиалов

## 🚀 Как начать использовать

### 1. Запустить проект
```bash
npm run dev
```

### 2. Войти в админ-панель
```
http://localhost:3000/admin/login
```

### 3. Открыть раздел "Менеджеры"
Нажать на пункт "Менеджеры" в боковой панели

### 4. Начать работу
- Добавить первого менеджера
- Назначить на филиал
- Протестировать все функции
