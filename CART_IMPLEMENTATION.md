# Реализация системы корзины

## Обзор

Реализована полноценная система корзины для пользователей с возможностью добавления товаров, управления количеством и оформления заказа.

## API Endpoints

### Корзина
- `GET /api/user/cart` - Получение корзины пользователя
- `POST /api/user/cart` - Добавление товара в корзину
- `DELETE /api/user/cart` - Очистка всей корзины

### Управление товаром
- `PATCH /api/user/cart/[id]` - Обновление количества товара
- `DELETE /api/user/cart/[id]` - Удаление товара из корзины

## Функционал

### 1. Добавление в корзину
- Кнопка "В корзину" на всех страницах с товарами:
  - Главная страница (`/home`)
  - Каталог (`/catalog`)
  - Детальная страница товара (`/product/[id]`)
- Индикатор загрузки при добавлении
- Автоматическое обновление счетчика корзины в header
- Если товар уже в корзине, увеличивается количество

### 2. Страница корзины (`/cart`)

**Список товаров:**
- Изображение товара
- Название, бренд, категория
- Артикул (SKU)
- Цена за единицу
- Управление количеством (+/-)
- Кнопка удаления товара
- Итоговая сумма за товар

**Итоговая панель:**
- Количество товаров
- Общая сумма
- Информация о доставке (самовывоз)
- Кнопка "Оформить заказ"

**Пустое состояние:**
- Иконка пустой корзины
- Сообщение "Корзина пуста"
- Кнопка "Перейти в каталог"

### 3. Счетчик корзины
- Отображается в header на всех страницах
- Автоматически обновляется при добавлении/удалении товаров
- Показывается только если есть товары в корзине

## Дизайн

**Цветовая схема:**
- Основной цвет: фиолетовый (#8b5cf6, #7c3aed)
- Фон: светло-серый градиент
- Карточки: белые с тенями

**Компоненты:**
- Карточки товаров с изображениями
- Кнопки управления количеством (+/-)
- Sticky итоговая панель справа
- Адаптивная сетка (1-2 колонки)

## База данных

**Таблица `carts`:**
```prisma
model Cart {
  id        String   @id @default(uuid())
  userId    String   @map("user_id")
  productId String   @map("product_id")
  quantity  Int      @default(1)
  createdAt DateTime @default(now())

  user      User     @relation(...)
  product   Product  @relation(...)

  @@unique([userId, productId])
}
```

**Особенности:**
- Уникальный индекс на (userId, productId)
- Cascade удаление при удалении пользователя или товара
- Автоматическое объединение товаров

## Безопасность

- Проверка авторизации (только для role=user)
- Валидация количества (минимум 1)
- Фильтрация только активных товаров
- Проверка принадлежности товара пользователю

## Файлы

**API:**
- `app/api/user/cart/route.ts` - Основные операции с корзиной

**Страницы:**
- `app/(user)/cart/page.tsx` - Серверный компонент
- `app/(user)/cart/CartClient.tsx` - Клиентский компонент

**Интеграция:**
- `app/(user)/home/HomeClient.tsx` - Кнопка "В корзину" на главной
