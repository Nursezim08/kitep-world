# Обновление: Уведомление вместо модального окна

## Изменения

Модальное окно для Google аккаунтов заменено на **всплывающее уведомление сверху страницы**.

## Новый дизайн

### Расположение
- ✅ Фиксированное положение сверху страницы
- ✅ По центру экрана
- ✅ Не блокирует весь экран
- ✅ Можно закрыть кнопкой X

### Анимация
```css
@keyframes slide-down {
  from {
    opacity: 0;
    transform: translate(-50%, -100%);
  }
  to {
    opacity: 1;
    transform: translate(-50%, 0);
  }
}
```

Уведомление плавно выезжает сверху за 0.3 секунды.

## Поведение

### Показ уведомления
Уведомление показывается когда:
1. Пользователь пытается войти по паролю с Google email
2. Пользователь пытается зарегистрироваться с существующим Google email

### Скрытие уведомления
Уведомление скрывается когда:
1. Пользователь нажимает кнопку X
2. Пользователь начинает вводить в форму
3. Пользователь нажимает "Войти через Google"

## Тестирование

### Тест 1: Показ уведомления
1. Зарегистрируйтесь через Google
2. Попробуйте войти с паролем
